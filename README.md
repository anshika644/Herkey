# Herkey

Herkey lead magnet prototype for a HerFuture salary calculator in partnership
with Simplilearn and HerKey.

## Run locally

No dependencies are required.

1. Open `index.html` directly in a browser, or
2. Serve the folder with any static file server.

## What it includes

- Single-screen salary calculator form:
  - Years of experience (dropdown)
  - Current/last role
  - Target role
- Practical salary potential estimate based on role transition and experience
- Non-selectable course-domain chip carousel (one chip visible at a time,
  auto-scrolling)
